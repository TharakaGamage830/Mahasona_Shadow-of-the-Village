# 🏮 MAHASONA: Shadows of the Village

**MAHASONA: Shadows of the Village** is a premium, logically-perfect Social Deduction Game inspired by Sri Lankan folklore and the mechanics of "Blood on the Clocktower". 

Set in the haunting village of Kandalama, players must identify the Mahasona posessing a villager before dawn, or face the demonic curse.

## ✨ Features

- **Folklore Inspired Roles**: 8 unique roles including the **Kattandiya** (Investigator), **Pirith Monk** (Healer), and the terrifying **Mahasona** himself.
- **Dynamic Lore**: Built-in instructions, grimoire, and story guides for an immersive experience.
- **Horror Aesthetics**: Custom typography, intense glow effects, and a persistent ritualistic music engine.
- **Global Authentication**: Secure login and registration powered by **Supabase**.
- **Real-time Gameplay**: Low-latency interactions using **Socket.io**.
- **Asset Protection**: Global protection for custom artwork and game assets.

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, Tailwind CSS, Framer Motion, Howler.js.
- **Backend**: Node.js, Express, Socket.io.
- **Database/Auth**: Supabase.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- Supabase Account

### Setup

1. **Clone the repository**
   ```bash
   git clone [your-repo-url]
   cd yaksha-game
   ```

2. **Backend Setup**
   ```bash
   cd server
   npm install
   # Create a .env file with your Supabase credentials
   npm run dev
   ```

3. **Frontend Setup**
   ```bash
   cd client
   npm install
   # Create a .env file with your Supabase credentials
   npm run dev
   ```
